# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-27

### Added
- 初始版本发布
- 支持自动脱敏 50+ 种敏感字段
- 支持 JSON、ENV、YAML 格式
- 支持自定义字段配置
- 纯 Shell 实现，零依赖
- 完整文档和使用指南

### Features
- `cat/less/grep` 命令自动脱敏
- 保留前后 4 个字符便于识别
- 原文件不被修改
- 一键安装/卸载
